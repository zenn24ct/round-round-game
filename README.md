# Color Switch

## 概要
- Canvas + JavaScript で作成したシンプルな Color Switch 風ゲーム
- ボールの色と同じ色のリング部分のみ通過可能
- コードむずかしくて、なんか難しいゲームになった。すぐゲームオーバーなる

## 操作方法
- クリック / タップ / Space：ジャンプ
- Game Over 時：同じ操作でリスタート

## ゲーム仕様
- ボールは重力で落下し、ジャンプで上昇
- リングは常に回転（ランダムなし・一定速度）
- ボールが下に移動したときのみ、障害物がスクロール
- 色チェンジャーに触れるとボールの色が変化
- スタート時に無敵時間あり（即死防止）

## 調整可能な主なパラメータ
- `OBSTACLE_RADIUS`：リングの大きさ
- `OBSTACLE_THICKNESS`：リングの太さ
- `OBSTACLE_GAP`：リング同士の間隔
- `ROT_SPEED_MULT` / `rotSpeed`：リングの回転速度
- `gravity`：重力の強さ
- `jump`：ジャンプ力
- `invincible`：開始時の無敵フレーム数

## 実行方法
- HTML ファイルをブラウザで直接開くだけ
- ローカルサーバー不要

## 開発メモ
- 衝突判定は少し甘めに設定
- タッチ操作対応（スマホ可）
- requestAnimationFrame を使用
